Welcome to the PanzerWeb all-purpose counter app! Our app is designed to help you keep track of any items that you need to count.

Please note that our counter is set up to avoid duplicate entries. This means that you cannot save the same number more than once within a single day. If you try to save a number that has already been saved, the app will alert you and prompt you to enter a different number. However, clicking the clear button will allow you to repeat the number

Double Click the clear button to remove all entries.

We have designed this feature to prevent errors and ensure the accuracy of your count. If you need to reset the count for a new day, simply close the app and reopen it.

We hope you find our app useful and easy to use. If you have any feedback or suggestions, please feel free to contact us at selwynvillar@gmail.com.

Thank you for using our app and have a great day!
